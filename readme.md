## Como instalar a aplicação localmente:

1. Para clonar a aplição:

    ```git clone https://github.com/brendatondin/randomTips```

2. Após clonar:

     ```cd randomTips```

3. Para instalar as dependências:

    ```npm i```

4. Para iniciar sua aplicação:

    ```npm start```
